# Appzinho FIAP 📱

Aplicativo mobile desenvolvido em React Native com Expo da FIAP.

## Requisitos

- Tela de Login com e-mail, senha e switch "lembrar-me"
- Busca por disciplina
- Componente personalizado `InputField`

## Tecnologias

- React Native
- Expo
- TypeScript
- StyleSheet

## Como rodar

```bash
npm install
npx expo start
```
